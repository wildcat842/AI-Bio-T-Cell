## 안전 망 (문제 발생 시)

| 증상 | 해결 |
| --- | --- |
| `mamba env create` 가 1시간+ 걸림 | Ctrl+C → `CONDA_SOLVER=libmamba mamba env create -f environment.yml` |
| `reticulate`가 시스템 python을 잡음 | `~/.Renviron`에 `RETICULATE_PYTHON=~/miniforge3/envs/AI-Bio-T-Cell/bin/python` |
| CUDA 버전 불일치 (`nvidia-smi`의 CUDA Version과 다름) | `environment.yml`의 `pytorch-cuda=12.4` → `12.1` 또는 `12.6`으로 변경 후 `mamba env update --prune` |
| `Seurat` 설치 실패 | `mamba install -n AI-Bio-T-Cell -c conda-forge r-seurat` 단독 시도 |
| `sceasy` GitHub 설치 실패 (의존성) | R 안에서 `BiocManager::install(c("LoomExperiment"))` 먼저 |
| 다운로드 중단 | 같은 명령 재실행 (aria2c가 자동 이어받기) |
| 디스크 부족 | Micro-C(`DataS7`)는 P2 tier - P0만 받음. `mamba clean -a -y` |
