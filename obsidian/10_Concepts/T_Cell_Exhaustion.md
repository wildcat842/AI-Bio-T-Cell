# T-Cell Exhaustion

## 정의
만성 항원 자극 하에서 effector function이 점진적으로 상실되고, 억제 수용체(PD-1, TIM-3, LAG-3 등)가 상승하는 CD8+ T cell의 분화 상태.

## 핵심 발견 (RIKEN Nomura 2025)
- "Exhausted" T cell은 **비가역적 죽은 상태가 아니다**.
- **Progenitor Exhausted (TPEX)** 서브셋이 self-renewal 능력을 가지며, ICI 반응의 핵심 책임 집단.
- 따라서 exhaustion은 **dynamic, regulatable 복잡계 상태**.

## 분화 trajectory (Daniel 2022)
```
Naive → Effector → Progenitor Exhausted (TPEX) → Terminal Exhausted (TEX)
                                  ↘ 자가갱신 (loop)
```

## 모델링 관점
- 4개 상태 = Waddington 경관의 어트랙터
- TPEX → TEX 전이 = 비가역 분기
- TPEX 자가갱신 = energy well의 깊이
